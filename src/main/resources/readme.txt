Essa API Rest foi criada com Java em Springboot, utilizando o banco H2 e anotações JPA para a montagem do banco e rotas.
A API está incompleta, porém está funcional. As chamadas existentes trarão o resultado de acordo com o que está descrito no controller.
Visto que não conseguimos chegar até ao ponto de interligar o front e o back, não existe implementação de autenticação para login, porém as primeiras iterações dessa funcionalidade estavam em desenvolvimento até o momento da entrega. Isso pode ser visto em "SecurityConfiguration.java"
